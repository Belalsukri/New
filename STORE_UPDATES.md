# تحديثات المتجر - Store Updates

## الميزات الجديدة المضافة

### 1. صورة البانر (Banner Image)
- تم إضافة إمكانية رفع صورة بانر كبيرة تظهر في أعلى صفحة المتجر
- الحجم الموصى به: 1200x400 بكسل
- تظهر الصورة بشكل جميل في أعلى صفحة تفاصيل المتجر
- عند وجود صورة بانر، يتم عرض شعار المتجر بشكل متداخل فوق البانر لمظهر احترافي

### 2. رقم الهاتف (Phone Number)
- تم إضافة حقل لرقم هاتف صاحب المتجر
- يظهر رقم الهاتف في صفحة تفاصيل المتجر مع أيقونة هاتف
- الرقم قابل للنقر عليه (clickable) لإجراء مكالمة مباشرة على الأجهزة المحمولة

## التغييرات التقنية

### Backend Changes:

1. **نموذج المتجر (Store Model)**
   - إضافة حقل `bannerImage`: لتخزين صورة البانر
   - إضافة حقل `phoneNumber`: لتخزين رقم الهاتف

2. **Store Routes**
   - تحديث `PUT /api/stores/my`: لدعم حفظ البانر ورقم الهاتف
   - تحديث `POST /api/stores/admin`: لدعم إنشاء متاجر مع البانر ورقم الهاتف

### Frontend Changes:

1. **StoreSettings.jsx**
   - إضافة حقل رفع صورة البانر مع معاينة
   - إضافة حقل إدخال رقم الهاتف
   - تحديث state management لدعم الحقول الجديدة

2. **StoreDetails.jsx**
   - عرض صورة البانر في أعلى الصفحة
   - عرض رقم الهاتف مع أيقونة وإمكانية الاتصال
   - تحسين التصميم ليكون أكثر احترافية

## كيفية الاستخدام

### لصاحب المتجر:
1. انتقل إلى "إعدادات المتجر" من لوحة التحكم
2. قم برفع صورة البانر (اختياري)
3. أدخل رقم الهاتف (اختياري)
4. احفظ التغييرات

### للزوار:
- عند زيارة صفحة المتجر، سيشاهدون:
  - صورة البانر الكبيرة في الأعلى (إن وجدت)
  - شعار المتجر
  - معلومات المتجر بما في ذلك رقم الهاتف القابل للنقر
